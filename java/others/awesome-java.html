                                                                                                                      # Java资源大全中文版

                                                                                                                      Awesome 系列虽然挺全，但基本只对收录的资源做了极为简要的介绍，如果有更详细的中文介绍，对相应开发者的帮助会更大。这也是我们发起这个开源项目的初衷。

                                                                                                                      # 古董级工具

                                                                                                                      这些工具伴随着Java一起出现，在各自辉煌之后还在一直使用。

                                                                                                                      • Apache Antopen in new window：基于XML的构建管理工具。官网open in new window
                                                                                                                      • cglibopen in new window：字节码生成库。官网open in new window
                                                                                                                      • GlassFishopen in new window：应用服务器，由Oracle赞助支持的Java EE参考实现。官网open in new window
                                                                                                                      • Hudsonopen in new window：持续集成服务器，目前仍在活跃开发。官网open in new window
                                                                                                                      • JavaServer Facesopen in new window：Mojarra是JSF标准的一个开源实现，由Oracle开发。官网open in new window
                                                                                                                      • JavaServer Pagesopen in new window：支持自定义标签库的网站通用模板库。官网open in new window
                                                                                                                      • Liquibaseopen in new window：与具体数据库独立的追踪、管理和应用数据库Scheme变化的工具。官网open in new window

                                                                                                                      # 构建工具

                                                                                                                      构建及应用依赖关系处理工具。

                                                                                                                      • Apache Mavenopen in new window：Maven是一款声明式构建及依赖管理工具，采用约定优于配置方式进行管理。相对Apache Ant更推荐使用Maven，前者采用了过程式管理，维护相对困难。官网open in new window
                                                                                                                      • Bazelopen in new window：来自Google的构建工具，可以快速、可靠地构建代码。官网open in new window
                                                                                                                      • Gradleopen in new window：使用Groovy（非XML）进行增量构建，可以很好地与Maven依赖管理配合工作。官网open in new window
                                                                                                                      • Buckopen in new window：Facebook构建工具。官网open in new window

                                                                                                                      # 字节码操作

                                                                                                                      编程方式操作字节码的开发库。

                                                                                                                      • ASMopen in new window：通用底层字节码操作和分析开发库。官网open in new window
                                                                                                                      • Byte Buddyopen in new window：使用流式API进一步简化字节码生成。官网open in new window
                                                                                                                      • Bytemanopen in new window：在运行时通过DSL（规则）操作字节码进行测试和故障排除。官网open in new window
                                                                                                                      • Javassistopen in new window：一个简化字节码编辑尝试。官网open in new window

                                                                                                                      # 集群管理

                                                                                                                      在集群内动态管理应用程序的框架。

                                                                                                                      • PMDopen in new window：对源代码分析查找不良的编程习惯。官网open in new window
                                                                                                                      • SonarQubeopen in new window：通过插件集成其它分析组件，对过去一段时间内的数据进行统计。官网open in new window
